# Noita Tablet Variants

Customizable reskin for the tablet.  
Feel free to contribute your own skins!

## Installation

Subscribe to the mod on its [Steam Workshop page](https://steamcommunity.com/sharedfiles/filedetails/?id=3695580957).

## Development

```sh
git clone https://github.com/rosalinadev/noita-tablet-variants.git
cd noita-tablet-variants
npm install

# To build the zip in dist
npm run build
# Or build an unzipped version
npx nts b -A

# To test the mod on an isolated Noita instance
npx nts run
```

Refer to [noita-ts](https://github.com/necauqua/noita-ts) for more info.
